**CyberDOP** es una plataforma financiera interactiva con una estética **Cyberpunk Femme-Fatale** (Magenta & Cyan). Diseñada para el monitoreo en tiempo real de las 12 divisas más relevantes a nivel global frente al **Peso Dominicano (DOP)**, integrando análisis visual de tendencias y una arquitectura robusta.

---

## ✨ Características Principales

- **UI Cyberpunk Personalizada:** Interfaz de alto impacto con gradientes magenta, bordes neón y tipografía futurista (Orbitron).
- **Master Calculator:** Conversión síncrona inteligente que permite calcular montos masivos en todas las divisas desde un único punto de entrada.
- **Visualización de Tendencias:** Gráficos de área dinámicos utilizando **ApexCharts.js** para visualizar la fluctuación de los últimos meses.
- **Consumo de API Real-time:** Integrado con **ExchangeRate-API** para garantizar precisión en los datos financieros.
- **Diseño Responsive:** Optimizado para dispositivos móviles y estaciones de trabajo de alta resolución.
- **Protocolo de Respaldo:** Lógica de JavaScript con sistema *fallback* para garantizar funcionalidad incluso en fallos de conexión.

---

## 🛠️ Tech Stack

### Frontend & UI
- **Core:** JavaScript (Vanilla ES6+), HTML5, CSS3.
- **Estilos:** Bootstrap 5 + Custom Cyberpunk CSS.
- **Gráficos:** [ApexCharts](https://apexcharts.com/).
- **Bundler:** [Vite](https://vitejs.dev/) (Build ultra-rápido).

### Herramientas y APIs
- **Data Source:** [ExchangeRate-API](https://www.exchangerate-api.com/).
- **Icons:** Font Awesome 6 + FlagCDN.
